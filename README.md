Project setup:

```zsh
$ pnpm install
$ cd claude-backend
$ pnpm install
$ touch .env
$ # copy the CLAUDE_API_KEY to .env
$ pnpm dev # start the backend server
$ # start another terminal
$ cd ..
$ pnpm dev
$ # open localhost:5173 in the browser

---
```
